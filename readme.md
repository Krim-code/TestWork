## How to build 

```
git clone https://github.com/Krim-code/TestWork.git
```


```
cd TestWork
```
### Create virtual env
```
python -m venv env
```
### Activate
```
sourse env/bin/activate

or

./env/Scripts/activate
```
### Install requirements
```
pip install -r requirements.txt
```
### How to start
```
python main.py
```
